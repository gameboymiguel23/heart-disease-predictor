# Heart Disease Predictor

## Beskrivning
Terminalbaserad AI-applikation som tränar en ML-modell på Heart Disease Dataset och gör prediktioner (target 0/1).

## Körning
1) Installera:
pip install -r requirements.txt

2) Kör:
python main.py

## Output
EDA-bilder sparas i outputs/:
- target_distribution.png
- correlation_heatmap.png
